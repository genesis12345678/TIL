# 자바 - 날짜와 시간

## Duration, Period

- **시간의 개념은 크게 2가지로 표현할 수 있다.**
  - 특정 시점의 시간(시각)
  - 시간의 간격(기간)
- `Duration`, `Period`는 시간의 간격(`amount of time`)을 표현하는데 사용된다.
